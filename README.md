# Typenimal 🐾⌨️

Welcome to **Typenimal**! This is a fast-paced, engaging typeracer game built with Unity. Test your keyboard skills, improve your typing speed, and challenge yourself across different game modes.

## 🎮 Play the Game

The game is published and ready to be played directly in your browser! 

👉 [**Play Typenimal on Unity Play**](https://play.unity.com/en/games/265201ad-a04d-439b-8106-3a636187b761/typenimalv1)

*(Note: For the best experience, please play on a desktop browser as typing games are not supported on mobile devices).*

## 🌟 Features

* **Classic Typeracer Mechanics:** Race against the clock to type out words accurately and quickly.
* **Multiple Game Modes:** Switch things up with different gameplay modes designed to test various aspects of your typing abilities.
* **Browser-Friendly:** Built with WebGL, meaning zero downloads or installations are required to jump into the action.

## 🛠️ Technologies Used

* **Game Engine:** Unity
* **Programming Language:** C#
* **Graphics & Shaders:** ShaderLab, HLSL
